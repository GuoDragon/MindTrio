# 操作记录

## 环境配置
1. 通过`python3 -m venv lcl`创建虚拟环境
2. 通过`source /home/ma-user/work/lcl/bin/activate`激活虚拟环境
3. 下载 0.5.1 版本的 MindSpore `pip install mindnlp==0.5.1`，他会直接安装相配套的 MindSpore、Transformers 等库
4. 下载 ipykernel `pip install ipykernel`，用于在 Jupyter Notebook 中运行 Python 代码
5. 注册当前虚拟环境的 Jupyter 内核 `python -m ipykernel install --user --name lcl --display-name "Python (lcl)"`
6. 在 Jupyter Notebook 中选择 `Kernel -> Change kernel -> Python (lcl)` 即可切换到当前虚拟环境
7. 注意每次重启 Notebook 后，都需要重新执行 2，4，5，6 步骤
